# My .tmux.conf

## Tmux version 2.1

### Status bar displays information for:

* hostname
* ip address
* color-changing battery percentage level for a laptop with one or two batteries
* connection to a VPN
* active/inactive windows
* current date
* current time
* current temperature
* current weather
